# 🎬 DudyFlix

> Plataforma pessoal de filmes e séries online 🍿

O **DudyFlix** é um projeto web inspirado em plataformas de streaming, desenvolvido com foco em praticar desenvolvimento front-end e organização de conteúdo multimídia.  
A aplicação permite adicionar, visualizar e organizar filmes e séries para assistir online de forma simples, moderna e intuitiva.
